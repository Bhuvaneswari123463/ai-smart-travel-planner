# 🌍 AI Smart Travel Planner

An intelligent travel planning web application that helps users plan trips efficiently using AI. It provides optimized routes, budget-friendly options, and crowd predictions for a better travel experience.

---

## 🚀 Features

### ✈️ Feature 1: Trip Planner

Users can plan their trips by entering:

* 📍 Start Location
* 🎯 Destination
* 💰 Budget
* 📅 Travel Dates
* ❤️ Interests

### 🤖 AI Generates:

* 🚆 Best transport routes
* 💸 Cheapest travel options
* ⏱️ Estimated travel time
* 🏨 Hotel suggestions
* 📍 Tourist attractions

#### 📌 Example:

**Input:**
Vijayawada → Goa
Duration: 3 days
Budget: ₹8000

**AI Output:**

* Bus: Vijayawada → Bangalore
* Train: Bangalore → Goa
* Hotel: Budget-friendly stays
* Places: Baga Beach, Fort Aguada

---

### 📊 Feature 2: Crowd Prediction (Unique Feature)

Predicts how crowded buses and trains will be using:

* 📈 Historical data
* 🤖 Machine Learning models

#### 📌 Example:

**Prediction:**

* Friday night buses → 🔴 High crowd
* Morning buses → 🟢 Low crowd

* Backend: Node.js / Express

## ⚙️ Installation & Setup

### 🔹 Backend

```bash
cd backend
npm install
npm run dev
```

### 🔹 Frontend

```bash
cd frontend/aiplanner
npm install
npm run dev
```

## 🔐 Environment Variables

Create a `.env` file in backend:

```

OPENROUTER_API_KEY=your_api_key
JWT_SECRET=your_secret
```
